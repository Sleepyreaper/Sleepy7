# Sleepy7 Monetization Guide

**Document type:** User guide  
**Audience:** Players, parents or guardians, customer support  
**Version:** 1.0  
**Last updated:** March 2026

## Overview

Sleepy7 is free to play. The core card game is available without a purchase.

This guide explains:

- what can be purchased
- where ads may appear
- how rewards work
- how parental controls affect purchases and ads
- what privacy choices apply to monetization features

## What you can buy

| Item | Type | What it does |
|---|---|---|
| Ad-Free | One-time purchase | Removes supported ads from the app |
| Premium Themes | One-time purchase | Unlocks additional visual themes |

These purchases are **non-consumables**, which means you buy them once and can restore them later through Apple’s purchase systems when available.

## What you do not need to buy

You do **not** need to make a purchase to:

- play the main game
- finish a round
- access basic gameplay features
- read the rules
- view your local stats
